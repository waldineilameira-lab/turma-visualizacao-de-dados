# 📊 Turma de Visualização de Dados

Repositório oficial da turma de Visualização de Dados. Aqui você encontrará materiais das aulas, exercícios, datasets e os projetos desenvolvidos pelos alunos.

## 📁 Estrutura do Repositório

```
turma-visualizacao-dados/
├── aulas/          ← Materiais e notebooks das aulas
├── exercicios/     ← Exercícios propostos
├── datasets/       ← Conjuntos de dados utilizados
├── alunos/         ← Repositórios e projetos dos alunos
└── projetos/       ← Projetos finais da turma
```

## 🚀 Como Participar

1. Faça um **fork** deste repositório
2. Clone o fork na sua máquina
3. Crie uma pasta com seu nome em `alunos/`
4. Desenvolva seus projetos e exercícios
5. Abra um **Pull Request** seguindo o template disponível

Consulte o [CONTRIBUTING.md](CONTRIBUTING.md) para mais detalhes.

## 📌 Regras

- Mantenha seus arquivos organizados dentro da sua pasta em `alunos/`
- Não modifique arquivos de outros alunos
- Use nomes de arquivo descritivos, sem espaços (use `_` ou `-`)
- Datasets grandes devem ser adicionados apenas na pasta `datasets/` compartilhada

## 📬 Dúvidas

Abra uma **Issue** neste repositório descrevendo sua dúvida.
