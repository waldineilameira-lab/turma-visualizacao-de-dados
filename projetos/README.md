# Projetos Finais

Galeria dos projetos de análise de dados desenvolvidos e aprovados pelos alunos da turma.

---

## O que é o Projeto Final?

Cada projeto é uma análise completa e autoral sobre um tema real, integrando todas as habilidades do curso:

| Etapa | Descrição |
|-------|-----------|
| **Coleta** | Obtenção de um dataset real de fonte pública ou aberta |
| **Limpeza** | Tratamento de valores nulos, duplicatas e inconsistências com Pandas |
| **Análise Exploratória** | Estatísticas descritivas, correlações e identificação de padrões |
| **Visualização** | Gráficos claros e bem documentados com Matplotlib, Seaborn ou Plotly |
| **Conclusões** | Insights baseados em evidências dos dados |

## Como publicar seu projeto

1. Desenvolva dentro da **sua pasta de aluno**: `alunos/seu-nome/projetos/projeto_final/`
2. Inclua um `README.md` com: descrição do tema, dataset utilizado, metodologia e principais descobertas
3. Garanta que todos os notebooks rodam sem erros
4. Abra um **Pull Request** — após revisão do professor, o projeto será adicionado à galeria acima

## Critérios de avaliação

| Dimensão | Peso | O que é avaliado |
|----------|------|-----------------|
| Qualidade dos dados | 20% | Limpeza, tratamento e documentação do dataset |
| Análise | 30% | Profundidade, relevância e correção das análises |
| Visualizações | 30% | Clareza, estética e adequação dos gráficos |
| Documentação | 20% | README, comentários no código e conclusões |
