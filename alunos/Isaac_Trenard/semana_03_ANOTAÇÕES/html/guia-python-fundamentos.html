Passo 3: Variáveis

Guardando informações na memória

💡 O que é uma variável?

Variável é como uma "caixa" onde você guarda um valor. Essa caixa tem um nome (identificador) e pode guardar diferentes tipos de dados: números, textos, etc.

variaveis.py
1# Tipos de dados básicos
2
3# Números inteiros (int)
4idade = 25
5ano = 2026
6
7# Números decimais (float)
8altura = 1.75
9preco = 19.90
10
11# Texto (string)
12nome = "João Silva"
13cidade = "São Paulo"
14
15# Verdadeiro/Falso (bool)
16estudante = True
17trabalha = False
18
19# Exibir valores
20print("Nome:", nome)
21print("Idade:", idade)
22print("Altura:", altura)

Regras para Nomes de Variáveis

✅ CORRETO
nome
idade_usuario
preco_produto
nota1
total_vendas
❌ ERRADO
1nome # não pode começar com número
meu-nome # não pode ter hífen
meu nome # não pode ter espaço
for # palavra reservada
Meu@Email # caracteres especiais
📝 Convenção: snake_case

Em Python, use snake_case para variáveis:
• Tudo minúsculo
• Palavras separadas por _ (underscore)
• Exemplos: nome_completo, preco_total, idade_usuario

Passo 5: Condicionais (if/else)

Tomando decisões no código

🤔 O que são condicionais?

Condicionais permitem que o programa tome decisões baseadas em condições.
É como perguntar: "SE isso for verdadeiro, faça aquilo. SENÃO, faça outra coisa."

condicional.py
1# Exemplo 1: Verificar maioridade
2idade = int(input("Digite sua idade: "))
3
4if idade >= 18:
5 print("Você é maior de idade")
6else:
7 print("Você é menor de idade")
8
9# Exemplo 2: Múltiplas condições (elif)
10nota = float(input("Digite sua nota: "))
11
12if nota >= 9.0:
13 print("Excelente!")
14elif nota >= 7.0:
15 print("Bom!")
16elif nota >= 5.0:
17 print("Regular")
18else:
19 print("Insuficiente")
⚠️ ATENÇÃO: Indentação é OBRIGATÓRIA!

Em Python, os espaços no início da linha NÃO são opcionais!
Use sempre 4 espaços (ou 1 tab) para cada nível de indentação.

Isso está ERRADO:
if idade >= 18:
print("Maior de idade") ← SEM indentação = ERRO!

Isso está CERTO:
if idade >= 18:
    print("Maior de idade") ← COM indentação

Operadores de Comparação

== (igual)

5 == 5 → True
5 == 3 → False

!= (diferente)

5 != 3 → True
5 != 5 → False

> (maior)

10 > 5 → True
3 > 10 → False

< (menor)

3 < 10 → True
10 < 5 → False

>= (maior ou igual)

5 >= 5 → True
3 >= 5 → False

<= (menor ou igual)

5 <= 5 → True
10 <= 5 → False

Passo 6: Loops (Repetição)

Repetindo ações sem copiar código

Loop FOR (repetições definidas)

loop_for.py
1# Repetir 5 vezes
2for i in range(5):
3 print("Contagem:", i)
4
5# Saída: 0, 1, 2, 3, 4
6
7# Tabuada do 5
8for i in range(1, 11): # 1 até 10
9 resultado = 5 * i
10 print(f"5 x {i} = {resultado}")

Loop WHILE (repetições condicionais)

loop_while.py
1# Contador de 0 a 4
2contador = 0
3
4while contador < 5:
5 print("Contador:", contador)
6 contador += 1 # contador = contador + 1
7
8# Validação de senha
9senha_correta = "1234"
10senha = ""
11
12while senha != senha_correta:
13 senha = input("Digite a senha: ")
14 if senha != senha_correta:
15 print("Senha incorreta!")
16
17print("Acesso concedido!")
⚠️ Cuidado: Loop Infinito!

Se a condição do while NUNCA ficar falsa, o programa nunca para!

while True:
    print("Infinito!")

Isso trava seu programa! Sempre tenha uma forma de sair do loop.
(Use Ctrl+C para parar um programa travado)
