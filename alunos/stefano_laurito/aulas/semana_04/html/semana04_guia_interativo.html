📌 Merge = JOIN do SQL
Une dois DataFrames pela coluna em comum (on=). Por padrão: INNER JOIN — só mantém linhas onde o ID existe nos dois lados. Para manter linhas sem par, use how='left'.
📌 Selecionar antes de juntar
Sempre selecione apenas as colunas necessárias das tabelas dimensão antes do merge. Evita DataFrames com dezenas de colunas inúteis e melhora performance.
📌 encoding='latin1' — quando usar?
Tabelas de cadastro com nomes brasileiros (ã, ç, é) precisam de encoding='latin1'. A tabela de vendas com apenas números e datas geralmente não precisa.
import pandas as pd

# Carregar todas as bases

lojas_df    = pd.read_csv(r'Contoso - Lojas.csv', sep=';', encoding='latin1')
clientes_df = pd.read_csv(r'Contoso - Clientes.csv', sep=';', encoding='latin1')

# Selecionar colunas necessárias ANTES do merge
clientes_df = clientes_df[['ID Cliente', 'E-mail']]
produtos_df = produtos_df[['ID Produto', 'Nome do Produto']]
lojas_df    = lojas_df[['ID Loja', 'Nome da Loja']]

# Merges encadeados (= JOINs do SQL)
vendas_df = vendas_df.merge(produtos_df, on='ID Produto')
vendas_df = vendas_df.merge(lojas_df,    on='ID Loja')
vendas_df = vendas_df.merge(clientes_df, on='ID Cliente')

# Renomear coluna ambígua

display(vendas_df)

📌 groupby — o GROUP BY do pandas
df.groupby('coluna').sum() agrupa linhas com mesmo valor e aplica uma agregação (sum, mean, count...). É a ferramenta central de qualquer análise por categoria.
📌 value_counts() vs groupby()
value_counts(): conta frequência de valores. Para "quantas compras cada cliente fez?"
groupby() + sum(): agrega valores numéricos. Para "quanto cada loja vendeu?"
📌 idxmax() — nome do maior
.max() → o maior valor numérico. .idxmax() → o rótulo (nome da linha) que tem o maior valor. Quando o índice é o nome da loja, idxmax() retorna o nome diretamente.
# Qual cliente comprou mais vezes?

display(frequencia_clientes)

vendas_lojas = vendas_df.groupby('Nome da Loja').sum()

# Ordenar e plotar top 5
vendas_lojas = vendas_lojas.sort_values('Quantidade Vendida', ascending=False)
vendas_lojas[:5].plot(figsize=(15, 5), kind='bar')

# Melhor loja
maior_valor = vendas_lojas['Quantidade Vendida'].max()
melhor_loja = vendas_lojas['Quantidade Vendida'].idxmax()

🔴 format='%d/%m/%Y' é obrigatório

Sem format, o pandas tenta inferir e pode errar datas brasileiras como 31/01/2017, gerando NaT (Not a Time). Sempre especifique o formato.

📌 Por que converter datas?
Data como string (object): não filtra por intervalo, não extrai mês/ano, não calcula diferença entre datas. pd.to_datetime() transforma em objeto real habilitando todas essas operações.
📌 loc vs iloc
loc[rótulo] — usa o nome do índice. Ideal quando o índice tem significado (nome de produto).
iloc[posição] — usa número inteiro. iloc[0] = sempre a 1ª linha.
Produção: prefira filtro booleano com loc: df.loc[df['ID'] == 873, 'Preco'] = 23

🔴 Erro clássico: 'and' em vez de '&'

Em pandas, and gera ValueError. Use sempre & (AND) e | (OR) com parênteses em cada condição:

# ERRADO — ValueError
df[df['ID'] == 306 and df['Qtd'] == 0]
# CORRETO
df[(df['ID'] == 306) & (df['Qtd'] == 0)]
📌 Como funciona o filtro
df['ID Loja'] == 306 → Series de True/False para cada linha
df[serie_booleana] → mantém só as linhas True

Salve a máscara em variável para facilitar debug:
mascara = df['ID Loja'] == 306
df_filtrado = df[mascara]
📌 '{:.2%}'.format() — formatação profissional
'{:.2%}'.format(0.0312) → 3.12% automaticamente. Multiplica por 100 e adiciona o símbolo. Use em todos os relatórios.
# % de devolução geral

⚠ Instalação necessária

✓ Versão vetorizada (sem loop)

Em produção: vendas_df.loc[vendas_df['ID Loja'] == 222, 'Quantidade Devolvida'] += 1 — uma linha, sem loop, muito mais rápido.

📌 Por que tqdm?
Loops sobre DataFrames grandes podem levar minutos. O tqdm adiciona barra de progresso com tempo decorrido e estimativa de conclusão — essencial em pipelines reais.
📌 enumerate() no loop
Retorna índice + valor de cada elemento. Com tqdm você tem: o índice da linha (para .loc[]), o valor e a barra de progresso — tudo junto.
from tqdm import tqdm

pbar = tqdm(total=len(vendas_df['ID Loja']), position=0, leave=True)

for i, id_loja in enumerate(vendas_df['ID Loja']):
    pbar.update()
    if id_loja == 222:
        vendas_df.loc[i, 'Quantidade Devolvida'] += 1

pbar.close()

# Versão profissional (vetorizada)
vendas_df.loc[vendas_df['ID Loja'] == 222, 'Quantidade Devolvida'] += 1

  • Sei usar pbar.update() e pbar.close()
