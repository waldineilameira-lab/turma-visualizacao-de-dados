Bloco 2 · 20h40 Módulo datetime — trabalhando com datas

"Turma, a coluna Data_Admissao chegou como string. O pandas não sabe que '13/08/2024' é uma data — pra ele é só texto. Se a gente não converter, não consegue calcular há quantos anos alguém está na empresa, não consegue filtrar quem foi admitido nos últimos 2 anos, não consegue agrupar por trimestre. Data é dado, não texto. Vamos resolver isso agora."

O módulo datetime do Python — o que você precisa saber

datetime.date
Representa apenas a data: ano, mês, dia. Sem hora.
datetime.datetime
Data + hora completos. É o tipo que o pandas usa internamente.
datetime.timedelta
Diferença entre duas datas. Resultado de data2 - data1.
strptime
String → Parsa → date. Converte texto em objeto de data.
strftime
String → Formata. Converte data em texto formatado.
pd.to_datetime
Versão pandas: converte coluna inteira de strings para datetime64.
EXEMPLO 05Convertendo datas e calculando tempo de casa

Esse é o uso mais frequente de datetime em análise de RH: converter a data de admissão e calcular há quantos anos (e meses) cada funcionário está na empresa.

aula_01.ipynb · Célula 10Python
import pandas as pd
from datetime import date, datetime, timedelta

# ── 1. Entendendo strptime e strftime (módulo datetime puro) ──
data_texto = "13/08/2024"

# strptime: converte string → objeto datetime
# %d = dia com zero à esquerda | %m = mês | %Y = ano 4 dígitos
data_obj = datetime.strptime(data_texto, "%d/%m/%Y")
print(f"String original : '{data_texto}'")
print(f"Objeto datetime : {data_obj}")
print(f"Tipo            : {type(data_obj)}")

# strftime: converte objeto datetime → string formatada
formatado = data_obj.strftime("%B de %Y")   # %B = nome do mês
print(f"Formatado       : {formatado}")

# ── 2. timedelta: diferença entre datas ──────────────────────
hoje = datetime.today()
diferenca = hoje - data_obj          # resultado é um timedelta
print(f"\nHoje            : {hoje.strftime('%d/%m/%Y')}")
print(f"Diferença       : {diferenca.days} dias")
print(f"Em anos         : {diferenca.days / 365.25:.1f} anos")

# ── 3. Aplicando no DataFrame inteiro ────────────────────────
# pd.to_datetime converte a coluna inteira — muito mais eficiente
# do que aplicar strptime linha a linha com .apply()
df["Data_Admissao"] = pd.to_datetime(
    df["Data_Admissao"],
    format="%d/%m/%Y",
    errors="coerce"   # datas inválidas → NaT (não quebra o pipeline)
)

print(f"\nTipo após conversão: {df['Data_Admissao'].dtype}")
print(f"Valores NaT gerados: {df['Data_Admissao'].isna().sum()}")

# ── 4. Coluna de tempo de casa em anos ───────────────────────
hoje_ts = pd.Timestamp.today()
df["Tempo_Casa_Anos"] = (
    (hoje_ts - df["Data_Admissao"]).dt.days / 365.25
).round(1)

# ── 5. Resultado ─────────────────────────────────────────────
print("\nAmostra — Data convertida e tempo de casa:")
cols = ["Nome", "Data_Admissao", "Tempo_Casa_Anos"]
print(df[cols].head(6).to_string(index=False))

Por que isso importa

pd.to_datetime com errors="coerce" é o padrão de mercado: datas inválidas (digitação errada no sistema) viram NaT em vez de derrubar o pipeline inteiro. A divisão por 365.25 (e não 365) considera anos bissextos — detalhe que importa quando você calcula tempo de casa de funcionários com 10+ anos na empresa.

EXEMPLO 06Colunas derivadas de data + filtros por período

Com a data convertida, extraímos atributos e aplicamos filtros — as operações mais pedidas em relatórios de RH industrial: headcount por ano de admissão, veteranos na empresa, admissões nos últimos 12 meses.

aula_01.ipynb · Célula 11Python
import pandas as pd

# Garantir que a coluna já foi convertida (rode após Célula 4)

# ── 1. Extraindo componentes da data com .dt ─────────────────
# .dt é o acessor de datetime no pandas — dá acesso a ano, mês, etc.
df["Ano_Admissao"]  = df["Data_Admissao"].dt.year
df["Mes_Admissao"]  = df["Data_Admissao"].dt.month
df["Dia_Semana"]    = df["Data_Admissao"].dt.day_name()   # nome do dia
df["Trimestre"]     = df["Data_Admissao"].dt.quarter

print("Colunas derivadas criadas:")
cols = ["Nome", "Data_Admissao", "Ano_Admissao", "Mes_Admissao", "Trimestre"]
print(df[cols].head(4).to_string(index=False))

# ── 2. Faixa de tempo de casa (coluna categórica) ─────────────
def classificar_tempo(anos):
    if anos < 1:
        return "Menos de 1 ano"
    elif anos < 3:
        return "1 a 3 anos"
    elif anos < 5:
        return "3 a 5 anos"
    else:
        return "Mais de 5 anos"

df["Faixa_Tempo_Casa"] = df["Tempo_Casa_Anos"].apply(classificar_tempo)

print("\nDistribuição por faixa de tempo de casa:")
print(df["Faixa_Tempo_Casa"].value_counts().to_string())

# ── 3. Filtros por período ────────────────────────────────────
hoje  = pd.Timestamp.today()
um_ano_atras = hoje - pd.DateOffset(years=1)
cinco_anos_atras = hoje - pd.DateOffset(years=5)

# Admitidos nos últimos 12 meses
novos = df[df["Data_Admissao"] >= um_ano_atras]

# Veteranos — mais de 5 anos de casa
veteranos = df[df["Data_Admissao"] < cinco_anos_atras]

print(f"\nAdmitidos nos últimos 12 meses : {len(novos)} funcionários")
print(f"Veteranos (5+ anos de casa)    : {len(veteranos)} funcionários")

# ── 4. Admissões por ano — panorama histórico ─────────────────
print("\nAdmissões por ano:")
print(
    df.groupby("Ano_Admissao")
      .size()
      .rename("Admissoes")
      .sort_index()
      .to_string()
)

# ── 5. Salário médio dos veteranos vs novos ───────────────────
print("\nSalário médio:")
print(f"  Novos (últimos 12 meses): R$ {novos['Salario'].mean():,.2f}")
print(f"  Veteranos (5+ anos)     : R$ {veteranos['Salario'].mean():,.2f}")

Por que isso importa

pd.DateOffset(years=1) é mais correto que pd.Timedelta(days=365) para filtros de "último ano" — ele considera anos bissextos e meses com 28/30/31 dias. alunos/seunome/semana_04/aula_01.pyPython
"""
aula_01.py — Semana 04 · Dia 01
Leitura de arquivos (CSV, JSON, Excel) + Módulo datetime

Pipeline:
    1. Leitura do CSV via URL RAW
    2. Exploração básica
    3. Exportação para JSON e Excel
    4. Conversão e manipulação de datas
    5. Geração do dataset enriquecido

Autor  : seunome
Turma  : T2 — SENAI BI
"""

import pandas as pd
import json
from datetime import datetime

# ═══════════════════════════════════════════════════════════
# 1. CONFIGURAÇÃO
# ═══════════════════════════════════════════════════════════

URL = (
    "https://raw.githubusercontent.com/"
    "cfneves/turma-visualizacao-de-dados/"
    "master/aulas/semana_04/base_rh.csv"
)

# ═══════════════════════════════════════════════════════════
# 2. EXTRAÇÃO — leitura do CSV com parâmetros corretos
# ═══════════════════════════════════════════════════════════

df = pd.read_csv(URL, sep=";", encoding="cp1252", decimal=",")
print(f"[1] CSV carregado: {df.shape[0]} linhas × {df.shape[1]} colunas")

# ═══════════════════════════════════════════════════════════
# 3. EXPLORAÇÃO BÁSICA
# ═══════════════════════════════════════════════════════════

print("\n[2] Tipos de dados:")
print(df.dtypes)

print("\n[3] Nulos por coluna:")
print(df.isnull().sum())

print("\n[4] Estatísticas numéricas:")
print(df.describe().round(2))

print("\n[5] Valores únicos por coluna:")
print(df.nunique())

print("\n[6] Distribuição — Departamento:")
print(df["Departamento"].value_counts())

# ═══════════════════════════════════════════════════════════
# 4. EXPORTAÇÃO — JSON e Excel
# ═══════════════════════════════════════════════════════════

# JSON
df.to_json("base_rh.json", orient="records", force_ascii=False, indent=2)
print("\n[7] base_rh.json gerado")

# Excel — uma aba por departamento
with pd.ExcelWriter("base_rh_deptos.xlsx", engine="openpyxl") as writer:
    df.to_excel(writer, sheet_name="Completo", index=False)
    for depto in sorted(df["Departamento"].unique()):
        df[df["Departamento"] == depto].to_excel(
            writer, sheet_name=depto[:31], index=False
        )
print("[8] base_rh_deptos.xlsx gerado")

# ═══════════════════════════════════════════════════════════
# 5. TRANSFORMAÇÃO DE DATAS
# ═══════════════════════════════════════════════════════════

# Converter string → datetime
df["Data_Admissao"] = pd.to_datetime(
    df["Data_Admissao"], format="%d/%m/%Y", errors="coerce"
)
print(f"\n[9] Data_Admissao convertida: {df['Data_Admissao'].dtype}")
print(f"    NaT gerados: {df['Data_Admissao'].isna().sum()}")

# Colunas derivadas
hoje = pd.Timestamp.today()
df["Ano_Admissao"]    = df["Data_Admissao"].dt.year
df["Mes_Admissao"]    = df["Data_Admissao"].dt.month
df["Trimestre"]       = df["Data_Admissao"].dt.quarter
df["Tempo_Casa_Anos"] = ((hoje - df["Data_Admissao"]).dt.days / 365.25).round(1)

def classificar_tempo(anos):
    if   anos < 1: return "Menos de 1 ano"
    elif anos < 3: return "1 a 3 anos"
    elif anos < 5: return "3 a 5 anos"
    else:           return "Mais de 5 anos"

df["Faixa_Tempo_Casa"] = df["Tempo_Casa_Anos"].apply(classificar_tempo)
print("\n[10] Colunas derivadas criadas: Ano_Admissao, Mes_Admissao, Trimestre,")
print("      Tempo_Casa_Anos, Faixa_Tempo_Casa")

# ═══════════════════════════════════════════════════════════
# 6. ANÁLISE — tempo de casa por departamento
# ═══════════════════════════════════════════════════════════

media_depto = (
    df.groupby("Departamento")["Tempo_Casa_Anos"]
      .mean().round(1)
      .sort_values(ascending=False)
      .rename("Media_Anos")
      .reset_index()
)
print("\n[11] Tempo médio de casa por departamento:")
print(media_depto.to_string(index=False))

# ═══════════════════════════════════════════════════════════
# 7. LOAD — salvar dataset enriquecido
# ═══════════════════════════════════════════════════════════

df.to_csv("base_rh_dia01.csv", index=False, sep=";", encoding="utf-8-sig")
print("\n[12] base_rh_dia01.csv salvo — pronto para o Dia 02")
print("      Colunas finais:", df.columns.tolist())
print("\n✓ Pipeline do Dia 01 concluído.")

Por que separar notebook e script

O notebook (.ipynb) tem saídas embutidas, estado de variáveis e células fora de ordem — ele não é executável de forma confiável em produção. O script (.py) é linear, sem estado oculto, e pode ser chamado por agendadores (cron, Airflow, Power Automate) ou importado por outros módulos. No Dia 03, você vai criar um src/limpeza.py que importa funções de aula_01.py — é assim que pipelines reais são construídos.
