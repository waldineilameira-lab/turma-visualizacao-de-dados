1

Acessar o Repositório

Primeiro passo: entrar no repositório da turma

🌐 Abra o Navegador
1. Abra o navegador (Chrome, Firefox, Edge, etc.)
2. Acesse: https://github.com/cfneves/turma-visualizacao-de-dados
3. Faça login na sua conta do GitHub (se ainda não estiver logado)
github.com/cfneves/turma-visualizacao-de-dados

📚 turma-visualizacao-de-dados

Repositório oficial da turma Visualização de Dados e BI T2 - SENAI 2026

⚠️ Atenção

Certifique-se de estar no repositório CERTO:
github.com/cfneves/turma-visualizacao-de-dados

NÃO é o seu fork! É o repositório original do professor.

2

Navegar até "alunos"

Encontrar a pasta onde você vai criar sua pasta pessoal

📁 Clicar na Pasta "alunos"
1. Na página principal do repositório, você verá uma lista de pastas e arquivos
2. Procure pela pasta chamada "alunos"
3. Clique nela para entrar
github.com/cfneves/turma-visualizacao-de-dados
📂 Arquivos e pastas:
📁 alunos ← Clique aqui!
📁 aulas
📁 datasets
📁 exercicios
📄 README.md
💡 Dica

A pasta "alunos" pode já ter pastas de outros alunos dentro. Tudo bem! Você vai criar a SUA pasta junto com as deles.

3

Criar Pasta + Arquivo

Usando a interface do GitHub

➕ Adicionar Novo Arquivo
1. Dentro da pasta "alunos", procure o botão "Add file" no canto superior direito
2. Clique nele
3. Escolha "Create new file"
github.com/.../turma-visualizacao-de-dados/tree/main/alunos

Você está em: alunos/

↓ Clique aqui e depois em "Create new file"

📝 Digitar o Caminho do Arquivo
No campo "Name your file", digite o caminho completo seguindo este formato:

seu_nome/README.md

Exemplo: Se seu nome é João Silva, digite:
joao_silva/README.md
🚫 IMPORTANTE: Regras de Nomenclatura

• Use snake_case: tudo minúsculo, separado por underscore (_)
• NÃO use espaços: joao silva ❌
• NÃO use hífen: joao-silva ❌
• NÃO use maiúsculas: JoaoSilva ❌
• USE underscore: joao_silva ✅

❌ ERRADO
João Silva/README.md
joao-silva/README.md
JoaoSilva/README.md
joaosilva/README.md
✅ CERTO
joao_silva/README.md
maria_santos/README.md
pedro_costa/README.md
ana_oliveira/README.md
4

Escrever o README

Adicionar informações básicas sobre você

✍️ Conteúdo do Arquivo
No editor que aparecer, digite o seguinte conteúdo (substitua com seus dados):
README.md
# João Silva
## Curso
Visualização de Dados e Business Intelligence T2
## Instituição
SENAI
## Sobre Mim
Estudante de Business Intelligence aprendendo Git, Python e análise de dados.
## Exercícios
- [ ] Exercício 01 - Git/GitHub
- [ ] Exercício 02 - Python Fundamentos
- [ ] Exercício 03 - Pandas Básico
💡 Personalize!

Você pode adicionar mais informações se quiser:
• Seus objetivos com o curso
• Linguagens que quer aprender
• Projetos que quer fazer
• Links para LinkedIn ou portfólio

💾 Fazer o Commit
1. Role a página até o final
2. No campo "Commit new file", escreva uma mensagem:
Cria pasta pessoal de João Silva
3. Deixe marcado "Commit directly to the main branch"
4. Clique no botão verde "Commit new file"
🎉 Parabéns!

Sua pasta foi criada no GitHub! Agora você vai ver:
alunos/seu_nome/README.md

A pasta aparece automaticamente porque você criou um arquivo dentro dela.

5

Clonar o Repositório

Baixar o repositório inteiro para seu computador

📥 Agora vamos trabalhar no PC

Sua pasta já existe no GitHub! Agora vamos baixar o repositório inteiro (com sua pasta) para o computador.

🔗 Copiar URL do Repositório
1. Volte para a página principal do repositório
2. Clique no botão verde "Code"
3. Copie a URL que aparece (HTTPS):
https://github.com/cfneves/turma-visualizacao-de-dados.git
Terminal / Git Bash
# Navegue até onde quer guardar o repositório
$ cd Documents
# Clone o repositório
$ git clone https://github.com/cfneves/turma-visualizacao-de-dados.git
# Aguarde o download...
# Entre na pasta
$ cd turma-visualizacao-de-dados
# Verifique que sua pasta está lá!
$ ls alunos/
✅ Verifique

Abra a pasta no explorador de arquivos e confira:
turma-visualizacao-de-dados/alunos/seu_nome/README.md

Seu arquivo está lá! 🎉

6

Trabalhar Localmente

Agora você pode adicionar arquivos e fazer commits normalmente

📁 Adicionar Arquivos na Sua Pasta
Exemplo: Criar pasta para Exercício 02 (Python)
Terminal
# Entre na sua pasta
$ cd alunos/seu_nome
# Crie pasta do exercício
$ mkdir exercicio_02
$ cd exercicio_02
# Crie seu arquivo Python
$ code caixa.py
💾 Fazer Commit e Push
Depois de criar seus arquivos:
Terminal
# Volte para raiz do repositório
$ cd ../..
# Veja o que mudou
$ git status
# Adicione seus arquivos
$ git add .
# Faça commit
$ git commit -m "[Seu Nome] Adiciona exercício 02 - Python"
# Envie para o GitHub
$ git push origin main
⚠️ IMPORTANTE

Você está fazendo push DIRETO no repositório principal!
NÃO está usando Fork → Pull Request.

Regra de ouro: Só mexa em alunos/seu_nome/
NUNCA altere arquivos fora da sua pasta!
