# Alunos

Cada aluno deve criar uma pasta com seu nome neste diretório.

## Estrutura esperada

```
alunos/
└── nome-sobrenome/
    ├── README.md       ← Apresentação (opcional)
    ├── exercicios/     ← Exercícios realizados
    └── projetos/       ← Projetos desenvolvidos
```
