Conceitos Básicos

Fundamentos que você precisa conhecer antes de começar

1

O que é Python?

Python é uma linguagem de programação de alto nível (próxima da linguagem humana), interpretada (não precisa compilar) e multiparadigma (permite diferentes estilos de programação). Criada por Guido van Rossum em 1991, o nome vem do grupo de comédia Monty Python.

2

Interpretada vs Compilada

Linguagem Compilada (C, Java): Código é traduzido para linguagem de máquina ANTES de executar. Mais rápido, mas precisa compilar toda vez que mudar o código.

Linguagem Interpretada (Python, JavaScript): Código é traduzido linha por linha DURANTE a execução. Mais lento, mas muito mais fácil de testar e debugar.

3

Indentação é OBRIGATÓRIA

Diferente de outras linguagens que usam {} para blocos de código, Python usa indentação (espaços ou tabs). Isso força o código a ser limpo e organizado. Regra: Use sempre 4 espaços (ou 1 tab, mas seja consistente).

4

Tipagem Dinâmica

Python é dinamicamente tipado: você não precisa declarar o tipo da variável. Python descobre sozinho! x = 5 (inteiro), x = "texto" (string) - a mesma variável pode mudar de tipo.

5

Case-Sensitive

Python diferencia maiúsculas de minúsculas. nome, Nome e NOME são três variáveis DIFERENTES! Convenção: use snake_case para variáveis (meu_nome) e PascalCase para classes (MinhaClasse).

Característica Python Outras Linguagens
Sintaxe Simples e legível, parece pseudocódigo Mais complexa, muitos símbolos
Declaração de Variável x = 5 int x = 5; (Java/C)
Blocos de Código Indentação (espaços) Chaves { } (Java/C/JS)
Fim de Linha Não precisa de ; (ponto e vírgula) Precisa de ; no final
Aprendizado Fácil para iniciantes Curva de aprendizado maior
⚠️ Atenção: Versão do Python

Existem duas versões principais: Python 2 (descontinuado em 2020) e Python 3 (atual). SEMPRE use Python 3! Para verificar: abra o terminal e digite python --version ou python3 --version.

Variáveis e Tipos de Dados

Como guardar e manipular informações

🔢

int (inteiro)

Números inteiros positivos ou negativos
idade = 25
temperatura = -10
ano = 2026

📊

float (decimal)

Números com ponto decimal
preco = 19.90
altura = 1.75
nota = 8.5

📝

str (string/texto)

Texto entre aspas (simples ou duplas)
nome = "João"
cidade = 'São Paulo'
mensagem = "Olá!"

✅

bool (booleano)

Verdadeiro ou Falso
ativo = True
pago = False
maior_idade = True

Operadores Matemáticos

Operador Descrição Exemplo Resultado
+ Adição 10 + 5 15
- Subtração 10 - 5 5
* Multiplicação 10 * 5 50
/ Divisão (retorna float) 10 / 5 2.0
// Divisão inteira 10 // 3 3
% Resto da divisão (módulo) 10 % 3 1
** Potenciação 2 ** 3 8
💡 Dica: Concatenação de Strings

O operador + também serve para juntar textos:
nome = "João"
sobrenome = "Silva"
nome_completo = nome + " " + sobrenome
Resultado: "João Silva"

⚠️ Cuidado: Loop Infinito!

Se a condição do while NUNCA ficar falsa, o loop nunca para!

while True:
    print("Infinito!")

Isso trava o programa! Sempre tenha uma forma de sair do loop.
